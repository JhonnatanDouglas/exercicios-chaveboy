Modulo 3 - Front React - Sprint 2

Nu Kenzie

Crie uma aplicacao React para controlar entradas e despesas. O usuario cadastra valores, ve o total e remove itens da lista.

ReactuseStatepropsstyled-components opcional

Dependencias principais

  • react e react-dom: base da aplicacao.
  • vite e @vitejs/plugin-react: criacao e servidor do projeto.
  • styled-components: opcional, para estilizar como componentes.
  • uuid ou crypto.randomUUID(): para gerar ids dos lancamentos.

Referencia visual

Use este link apenas como guia de layout: Nu Kenzie publicado.

Voce nao precisa copiar pixel por pixel. Precisa entregar a mesma ideia: formulario de cadastro, resumo de total e lista de lancamentos.

Ordem de montagem

  1. Crie o projeto com Vite e limpe a estrutura inicial.
  2. Crie o layout base: header no topo, formulario na esquerda e lista na direita.
  3. No App, crie o estado transactions como array vazio.
  4. Crie o formulario com descricao, valor e tipo. Primeiro faca ele capturar dados, depois faca ele enviar.
  5. No envio, valide os campos. Se estiver tudo certo, crie um objeto com id, description, value e type.
  6. Adicione o novo objeto no estado sem alterar o array original.
  7. Crie o componente de total e calcule: entrada soma, despesa subtrai.
  8. Crie a lista de cards e renderize cada lancamento.
  9. Crie o botao remover e filtre o item pelo id.
  10. Finalize responsivo: no celular, formulario e lista ficam um abaixo do outro.

Regras de aceite

  • O campo descricao e obrigatorio.
  • O campo valor deve ser numerico e maior que zero.
  • O tipo deve ser apenas entrada ou despesa.
  • Cada lancamento precisa ter um id unico. Pode usar crypto.randomUUID().
  • O total deve atualizar automaticamente ao adicionar ou remover itens.
  • O estado principal deve ficar no componente pai e ser enviado por props para os filhos.

Componentes sugeridos

  • Header: mostra o nome da aplicacao.
  • Form: cadastra descricao, valor e tipo.
  • TotalMoney: recebe a lista e calcula o total.
  • FinanceList: decide se mostra lista vazia ou cards.
  • FinanceCard: mostra um item e o botao remover.
